# Foehammer's Python Sandbox

## Local Development

### Prerequisites

- [Install uv](https://docs.astral.sh/uv/getting-started/installation/)
    - ```bash
      uv sync
      ```
    - ```bash
      uv run pre-commit install
      ```
- System Dependencies:
    - PDF Tools
        - [Tesseract OCR](https://github.com/tesseract-ocr/tesseract)
